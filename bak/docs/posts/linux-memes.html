自製的Linux梗圖分享

2026-05-24

接下來這些可能會有點離譜，

大家坐穩了！

提示：我使用的是Meee圖床，在國內會有更好的體驗

Chroot Overdose

笑點：

Arch Linux安裝時需要使用到chroot。

執行後，會將根目錄(/)設定成指定的位置（e.g. /mnt/）

而這裡便是使用了《主播女孩重度依賴》的二創圖進行三創。

Arch 超勇的好不好

笑點：

Arch Linux安裝一般比較硬核，大多Arch使用者會以此自滿。

但旁邊的Gentoo會從原始碼開始進行編譯，

看來是要被登dua郎了（怕

GNU/Linux

笑點：

RMS指的是Richard Stallman，自由軟體基金會創始人

當時的FSF地下的GNU計畫做了一堆工具沒內核；Linux只有內核沒工具

後來就合而為一，RMS認為只叫Linux讓GNU計畫看起來像空氣，硬要叫GNU/Linux

Manjaro RTFM

笑點：

Manjaro是基於Arch的發行版，但是對新手友善

而Arch社群一直有個梗，RTFM(Read The F\\king Manual)

加上原生Arch使用者的自滿，導致這玩意被常常拿來對付問了制帳簡單問題的Manjaro使用者

Mint vs Ubuntu

笑點：

Mint是個對新手極為友善的發行版；Ubuntu是市占率極高的發行版

導致新手常常不知道選哪個好

仔細看會發現Linux前面有個超迷你的「GNU/」

rm -rf / It's fint

笑點：

`rm -rf /`是個經久不衰的Linux自毀指令，

一堆梗圖都引用這指令，包含這個

不要亂用，尤其是父母的電腦，到時候孝都孝不出來

Admin vs root

笑點：

Windows的管理員常常遇上權限或系統限制；

Linux是你有權限想毀滅世界都行

記住那句「能力越大，責任越大」

Windows Update 大戰 GRUB

笑點：

Windows老是覺得自己是唯一的系統

一更新就把Linux的開機引導程式GRUB拆了

我也是受害者，寫這篇的前一天才發現GRUB被炸了

Windows 11 vs Debian

笑點：

Windows 11的硬體要求讓很多人上不去

但多數人不買新電腦而是安裝Linux

然後很多電腦上Debian還比Windows 11穩定、流暢

以上便是我製作的梗圖，

希望能陪你度過這美好的一天
