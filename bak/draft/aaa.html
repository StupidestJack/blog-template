Linux 是什麼？

Linux 是一個 Unix-like 作業系統核心。
 與 Windows、macOS 等系統類似。
 相較於另外兩個，Linux 對普通使用者而言，
 有「免費」、「自由」跟「自定義能力強」三個優點。
  • 免費：這個應該就不用我解釋了吧。
  • 自由：Linux 允許使用者替換系統核心，你不一定要像 Windows 或 macOS 全盤接受更新。你可以選擇魔改核心（如linux-zen）取得更好的效能；也可以選擇長期穩定版（如linux-lts）取得更穩定的體驗。
  • 自定義能力強：大多數 Linux 桌面環境都允許你修改主題。這個主題不只是改改配色的扮家家酒，而是徹底更改按鈕樣式的那種。

什麼是發行版？

發行版（Distro）有點類似Android上不同廠商的版本。
 比如三星的OneUI、小米的HyperOS等，
 因為理念、要求各不同，所以設計和功能上也不同。
 Linux的發行版在衍生關係上比較麻煩，它的發行版可以一個衍生一個，一個再衍生另一個。
 比方說Linux衍生出Debian，Debian再衍生出Ubuntu，Ubuntu再衍生出Linux Mint，Linux Mint還分三個版本。
 挑選適合自己的發行版也是很重要的。

我適合什麼發行版？

其實這篇文章是面向新手的。
 對於新手，千萬不要相信網友去裝Arch或者Gentoo這種硬到過分的發行版。
 雖然標題有Arch，但那是這個系列的長遠計畫。
 本人建議這些發行版：
  • Linux Mint（最推薦）：樣式很像Windows，轉移難度低，內建應用商店也很方便。
  • Ubuntu：老牌發行版，社群最豐富，對新手也很友善。
  • Zorin OS：標榜最像Windows，但某些功能要付費，本人不是很喜歡。
之後的系列裡，我會先以Linux Mint當最主要的教學發行版。
 此外也會有包含安裝、配置、日常使用、遊戲，到終端機、桌面環境、更深的發行版等等，未來會繼續更新。
